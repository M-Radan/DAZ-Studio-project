# DAZ Studio – Računalna grafika

Ovaj projekt izrađen je u sklopu kolegija **Računalna grafika** i prikazuje osnovne mogućnosti alata **DAZ Studio** kroz praktičnu izradu 3D scene i animacije.

## Opis projekta

Cilj projekta je upoznavanje s radom u DAZ Studiju, uključujući:
- kreiranje i prilagodbu 3D modela
- izradu scene
- animaciju lika
- postavljanje kamere i rasvjete
- završno renderiranje scene

## Korišteni alat
- **DAZ Studio**

## Zaključak

Projekt demonstrira praktičnu primjenu DAZ Studija u izradi 3D sadržaja te pruža osnovni uvid u proces modeliranja, animacije i renderiranja.
